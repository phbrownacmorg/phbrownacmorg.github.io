Font properties

CSS allows you to specify different font families: sans-serif, monospace, and cursive, as well as the default serif. You can also tweak within families. Font styles can be normal, italic, or oblique. Font variants can be normal or Small caps. Font size can be varied, as can font weight.

Text properties

Text properties applicable to entire paragraphs include text-indent and text-align. text-indent allows the first line of a paragraph to be indented a given amount from the left (more or less as you would expect). text-align allows text to be aligned:

left,

center,

or right.

Available text decorations include overline (as for a vector), underline (for emphasis), and line-through (for mistakes/edits).

text-transform allows text to be rendered in uPpErCaSe, title case (approximately), or LoWeRcAsE regardless of its actual capitalization. (Note that text-transform: capitalize does not change the capitalization of letters other than the first letter in each word.)

white-space gives several ways of making the white space in the HTML code significant in the output.
